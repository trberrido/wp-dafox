# wp-dafox

Tailored theme for https://dafox.co

## Features
### SEO
- OGP tags
- meta description
- canonical URL
- disable archives
### Performance
- basic page caching
- image: AVIF / WEBP formats
- disable auto feeds

### Security
- REST API: block user enumeration

### Misc
- basic multilanguage switch
- use the classic menu as a block

## Outside of WordPress scope
### Performances
- CDN
- OPCache
- http/2
### Security
- .htaccess:
	- disable xmlprc
	- limit access for wp-login.php
